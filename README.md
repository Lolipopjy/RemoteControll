# RemoteControll
* 手机与电脑在同一个局域网内，通过手机可以操控电脑
* 可以浏览电脑的文件
* 可以把电脑的文件下载到手机
* 可以操控电脑的鼠标
* 可以操控电脑的键盘
* 可以把手机的文件上传到电脑
